# How to play

## Generating map

Go to "Map Creation" tab
Use the sliders to set up your game
Click `Generate`
Select which tiles should have walls
Click `Save`

## Playing

After creating the map, go to the "Active Game"
Click `Load`
Play the game
After the game ends, click `Play again` to reset the game
